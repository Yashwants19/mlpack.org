radical

NAME

radical - radical

SYNOPSIS

radical [-h] [-v] --input_file string [--angles int] [--noise_std_dev double] [-o string] [-u string] [--replicates int] [--sweeps int]

DESCRIPTION

An implementation of RADICAL, a method for independentcomponent analysis (ICA). Assuming that we have an input matrix X, thegoal is to find a square unmixing matrix W such that Y = W * X and the dimensions of Y are independent components.

For more details, see the following paper:

@article{ title = {ICA Using Spacings Estimates of Entropy}, author = {Learned-Miller, E.G. and Fisher III, J.W.}, journal = {Journal of Machine Learning Research}, volume = {4}, pages = {1271--1295}, year = {2003}

REQUIRED OPTIONS

--input_file [string]

Input dataset filename for ICA.

OPTIONS

--angles [int]

Number of angles to consider in brute-force search during Radical2D. Default value 150.

--help (-h)

Default help info.

--info [string]

Get help on a specific module or option. Default value ’’.

--noise_std_dev [double]

Standard deviation of Gaussian noise Default value 0.175.

--output_ic (-o) [string]

File to save independent components to. Default value ’output_ic.csv’.

--output_unmixing (-u) [string]

File to save unmixing matrix to. Default value ’output_unmixing.csv’.

--replicates [int]

Number of Gaussian-perturbed replicates to use (per point) in Radical2D. Default value 30.

--sweeps [int]

Number of sweeps (each sweep calls Radical2D once for each pair of dimensions Default value 0.

--verbose (-v)

Display informational messages and the full list of parameters and timers at the end of execution.
