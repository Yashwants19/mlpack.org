NAME

local_coordinate_coding - lcc

SYNOPSIS

local_coordinate_coding [-h] [-v] --data string -l double -k int --n_iterations int [--initial_dictionary string] [--results_dir string]

DESCRIPTION

An implementation of Local Coordinate Coding

REQUIRED OPTIONS

--data [string]

path to the input data.

--lambda (-l) [double]

weighted l1-norm regularization parameter.

--n_atoms (-k) [int]

number of atoms in dictionary.

--n_iterations [int]

number of iterations for sparse coding.

OPTIONS

--help (-h)

Default help info.

--info [string]

Get help on a specific module or option. Default value ’’.

--initial_dictionary [string]

Filename for initial dictionary. Default value ’’.

--results_dir [string]

Directory for results. Default value ’’.

--verbose (-v)

Display informational messages and the full list of parameters and timers at the end of execution.
