mlpack
fast, flexible C++ machine learning library
written in C++ and built on the Armadillo linear algebra library, the ensmallen numerical optimization library, and parts of Boost.
aims to provide fast, extensible implementations of cutting-edge machine learning algorithms. Read the release announcement.
